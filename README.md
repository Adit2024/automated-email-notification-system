# Automated Email Notification System

## 📌 Overview
This project automates email notifications using Zapier. 
Whenever a new entry is added to Google Sheets, an email is automatically triggered via Gmail.

## ⚙ Workflow Architecture

Trigger:
- Google Sheets → New Spreadsheet Row

Action:
- Gmail → Send Email

## 🛠 Tools & Technologies
- Google Sheets
- Zapier
- Gmail

## 🎯 Objective
To eliminate manual email handling and implement real-time automation workflows.
